# 📊 Global Oil Price Trend & Volatility Analysis Dashboard

## 🔍 Overview
This project is an interactive Power BI dashboard analyzing global oil price trends and volatility over time.

---

## 📌 Key Features
- KPI metrics: Avg Price, Max Price, Min Price, Volatility %
- Brent oil price trend (time-series analysis)
- Year-over-Year (YoY) change to highlight volatility
- Interactive year slicer for filtering

---

## 📈 Key Insights
- Oil prices peaked during major global events (2008, 2012)
- High volatility observed in recent years
- Sharp drop in 2020 followed by recovery

---

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
